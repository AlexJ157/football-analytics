# Football Analytics

A Python project that uses football data from APIs to explore team and player statistics, analyse performance, and eventually build football match predictions.

## Project Goals

This project aims to build an interactive football analytics application that can:

- Retrieve football data using APIs
- Analyse team and player performance
- Display useful statistics and visualisations
- Explore historical match data
- Predict the outcomes of future matches using machine learning

## Technologies

- Python
- REST APIs
- Pandas
- Data visualisation
- Machine learning
